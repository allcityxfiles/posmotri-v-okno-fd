# Посмотри в окно

## Описание проекта

Веб-приложение для поиска и просмотра видеороликов с видами из окон различных городов в разное время суток. Пользователь может выбрать город и время суток (утро, день, ночь), после чего получает подборку соответствующих видео. Проект решает задачу виртуального "путешествия" через окна разных домов, позволяя увидеть атмосферу городов в разные периоды дня.

**Технологии:**
- HTML5 (семантическая разметка, template)
- CSS3 (Grid, Flexbox, кастомные свойства, псевдоэлементы)
- Чистый JavaScript (без фреймворков)
- Адаптивная вёрстка (фиксированная ширина 1200px)
- Кастомные чекбоксы и скроллбар
- Доступность (ARIA, скрытые элементы, focus-visible)

**Структура проекта:**

## Использование

1. **Поиск видео:**
   - Введите название города в поле "Город"
   - Выберите одно или несколько времён суток
   - Нажмите кнопку "Найти"

2. **Просмотр результатов:**
   - Основное видео отображается в левой части
   - Список похожих видео — в правой колонке
   - Для прокрутки списка используйте скролл
   - Нажмите на миниатюру для просмотра видео

3. **Особенности интерфейса:**
   - Кастомные чекбоксы с hover-эффектами
   - Focus-visible для доступной навигации
   - Анимации при взаимодействии
   - Прелоадер при загрузке данных
   - Сообщения об ошибках
